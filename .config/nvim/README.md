# Installation

brew install neovim

- For treesitter plugin
brew install tree-sitter

- For prettier
brew install prettierd

- For telescope 
brew install ripgrep

- For nvim-tree
https://www.nerdfonts.com/font-downloads

- For LSP
npm i -g typescript-language-server
